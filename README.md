# BioCellR8 LLC - Official Website

**Live Site:** https://pixelpolicyuser.github.io/biocellr8/

## About

Official website for BioCellR8 LLC, a translational sciences accelerator consultancy and biotech incubator based in Charlton, Massachusetts. 

BioCellR8 helps life sciences innovators bridge the gap between groundbreaking research and real-world commercial impact through expert regulatory affairs consulting, operational excellence, strategic advisory, and world-class incubator facilities.

## Technology Stack

- Pure HTML5, CSS3, and Vanilla JavaScript
- No external frameworks or paid services
- Fully responsive and accessible (WCAG 2.1 AA compliant where practical)
- Optimized for GitHub Pages deployment
- SEO-optimized with Schema.org structured data

## Design Philosophy

- **Scientific Credibility**: Clean, precise, evidence-based aesthetic
- **Innovation**: Modern teal and gold accents suggesting cutting-edge work
- **Trust**: Professional navy palette, clear typography, and generous white space
- **Action-Oriented**: Prominent, functional CTAs throughout

## Local Development

1. Clone the repository
2. Open `index.html` directly in any modern browser
3. Edit files as needed — changes reflect immediately (no build step required)

## Deployment

This site is **automatically deployed** to GitHub Pages whenever changes are pushed to the `main` branch.

[![Deploy BioCellR8 Website to GitHub Pages](https://github.com/pixelpolicyuser/biocellr8/actions/workflows/deploy.yml/badge.svg)](https://github.com/pixelpolicyuser/biocellr8/actions/workflows/deploy.yml)

**How it works:**
- Push to `main` (or manual trigger via **Actions** tab) → GitHub Actions workflow runs
- Site is validated and deployed to production in ~30-90 seconds
- Full deployment history and logs available in the Actions tab
